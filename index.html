4. Common Issues & Troubleshooting

4.1 Appointment Issues

Issue: Can't select clients

Solution: Make sure you've selected a Staff Member first. Client selection is disabled until a staff member is chosen.

Issue: No authorizations available

Solution: The selected client doesn't have active authorizations. Either:

  • Add authorizations for the client first
  • Switch to Non-Billable appointment
  • Select a different client

Issue: Can't create appointment

Solution: Check that all required fields (*) are filled:

  • Staff Member
  • Date
  • Start/End Time
  • Location
  • For Clinical: Client, Service
  • For Billable: Authorization

Issue: Appointment name not auto-filling

Solution: Make sure you've filled:

  • Start Time
  • End Time
  • (Optional) Client(s) for better naming
The name should update automatically as you fill these fields.

4.2 General Issues
